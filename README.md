App2 — Mini-Projetos Desenvolvidos em Flutter

Descrição:
O App2 é um conjunto de mini-projetos criados para estudo e prática de Flutter e Dart. O projeto reúne exemplos independentes, incluindo formulários, drawers, widgets Stateful e Stateless, além de interfaces simples para reforçar conceitos fundamentais do desenvolvimento mobile com Flutter.

Tecnologias:
Flutter, Dart, VS Code ou Android Studio.

Conteúdos / Mini-projetos incluídos:

- Formulário completo;
- Drawer funcional;
- Navegação entre telas apenas no módulo do formulário, permitindo acesso à tela de login;

Exemplos de:

- StatefulWidget
- StatelessWidget
- Telas simples para treino de layout;
- Componentes básicos e práticas de UI.

Como executar:

- Clone o repositório;
- Certifique-se de ter o Flutter instalado e configurado;

No terminal, execute:

flutter pub get

flutter run
